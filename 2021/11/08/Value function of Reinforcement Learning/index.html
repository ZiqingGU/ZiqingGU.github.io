Value Function of Reinforcement Learning

Posted on 2021-11-08 Edited on 2021-12-02 In Ph.D. Research

值函数本质是状态的函数，有两种表达方式：

一种是表格，另一种是带参的近似函数：
$$
v_(s;\omega )
$$
并利用样本学出函数的参数。

每次迭代都意味着值函数要向对应值函数的目标值接近一步。

蒙特卡洛方法里的状态价值与带参的近似值函数无关，是一种无偏估计。其他的类似TD的方法，状态价值与带参的近似值函数无关，此时，状态的价值估计就是一种有偏估计，一般的处理方式是半梯度方法(semi-gradient methods)，即在求导时忽略状态的价值估计和近似值函数的参数有关。半梯度方法一般收敛表现没那么稳健，但快很多，同时不需要等到一个事件结束才开始学习。在利用线性函数去逼近值函数的情况下，可证明半梯度方法是收敛的。
